War Night 41 proved that staggered relief rotations outperform static squadron assignments for extended defensive operations. This guide formalizes the approach for squadron leads planning their own rotations.

## The core idea

Instead of assigning each squadron to a single vector for the whole operation, rotate squadrons through a rest cycle so no vector is ever left with fatigued or thinned-out pilots.

## Rotation structure

- Divide available squadrons into three groups: **Active**, **Reserve**, and **Repair**
- Active groups hold vectors for one wave before rotating to Repair
- Repair groups rearm and re-shield before moving to Reserve
- Reserve groups move to Active for the next wave

## Coordination requirements

This only works with clear voice discipline. One tactical officer should own rotation calls exclusively — mixing rotation calls with combat calls on the same channel caused confusion in early tests.

## When not to use it

Staggered rotations assume an extended, multi-wave engagement. For short, decisive pushes, standard full-commitment doctrine is still faster.
